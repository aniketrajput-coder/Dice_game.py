# Dice_game.py
# Dice Game (Python)
A simple command-line Dice Game built using Python where two players compete over 3 rounds to get the highest score.

# Features
Two-player game
Each player gets 3 chances (rounds) to roll the dice
Random dice number between 1 and 6
Automatic score calculation
Winner announcement at the end
Draw condition handled 

 # How It Works
The game starts by asking names of two players.
Player 1 plays all 3 rounds first:
Press Enter to roll the dice
A random number (1–6) is generated
Score is added
Player 2 then plays 3 rounds in the same way.
Final scores are displayed.
The program compares scores and declares:
Winner 
Or Draw
